#  Portfolio Website Project 
This project is a personal portfolio website built using Astro, a modern web framework that allows for fast and efficient development of websites. The website features a responsive design, a carousel for showcasing projects, and a layout that includes a sidebar, header, and footer. The project utilizes various technologies such as Tailwind CSS for styling, Vite for building, and simple-icons for icons.

##  Features
* Responsive design for optimal viewing on various devices
* Carousel for showcasing projects
* Layout that includes a sidebar, header, and footer
* Utilizes Tailwind CSS for styling
* Built with Vite for fast and efficient development
* Includes simple-icons for icons
* Supports dark mode and language toggling

##  Tech Stack
* Astro: A modern web framework for building websites
* Tailwind CSS: A utility-first CSS framework for styling
* Vite: A fast and efficient build tool
* simple-icons: A library of icons for use in the project
* JavaScript: Used for scripting and dynamic functionality
* CSS: Used for styling and layout
* HTML: Used for structuring and content

##  Installation
To get started with the project, follow these steps:
1. Clone the repository using `git clone`
2. Install the dependencies using `npm install` or `yarn install`
3. Start the development server using `npm run dev` or `yarn dev`
4. Build the project using `npm run build` or `yarn build`

##  Usage
To use the project, simply navigate to the website in your web browser. The website is fully responsive and can be viewed on various devices.
